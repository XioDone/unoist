Unoist

⚡️ Make using UnoCSS simpler and easier.

Install

pnpm i unoist

Usage

  • Configure ESM as default in package.json , it's up to you 
{
          "type": "module"
        }
  • Add shortcuts to your uno.config.ts 
import { useShortcuts } from 'unoist'
        export default defineConfig({
          shortcuts: [
            // Your config ...
            ...useShortcuts(),
          ],
        })
